# Mewgenics Cat Visual Organizer

This project is a local web-based inspector for visualizing cat data from the game Mewgenics. It allows you to upload or load JSON files containing cat data and view them in a user-friendly interface. No backend is required; all data is handled client-side.

## Features

- Upload or load cat data from JSON files
- Visualize cat relationships and attributes
- No persistent storage or editing

## Project Structure

- `src/` — Frontend code (React, Vite)
- `get-the-data/` — Scripts and data files for extracting and preparing cat data
- `package.json` — Project dependencies and scripts
- `vite.config.js` — Vite configuration

1. Run `npm install` to install dependencies.
2. Start the local server with `npm run dev`.
3. Upload or load your cat data JSON file.
